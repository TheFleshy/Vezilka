# NLP Dataset Cleaner

Скриптата чита JSON/JSONL фајл, ги филтрира и зачувува само чист датасет за натамошна употреба во NLP на македонски јазик во JSONL фајл.

## Користени библиотеки
* `datatrove` (Pipeline рамка за обработка на текстуални датасети)
* `re` (regex за детектирање на кирилица)
* `json` (читање на JSON/JSONL фајлови)
* `shutil` / `os` (управување со фајл-систем и директориуми)

## Структура на кодот
Кодот е поделен на четири логички целини:

1. **UniversalReader — Читање на податоци**
   * Читачот ги поддржува и `.json` и `.jsonl` форматите.
   * Го извлекува полето `"text"` или `"content"`.
   * Ги отстранува HTML таговите со regex: `re.sub(r'<[^>]+>', '', text)`
   * Го враќа секој запис како `Document` објект.

2. **macedonian_filter — Јазичен филтер**
   * Проверува дали документот е претежно на македонски јазик.
   * Ги брои македонските кирилични букви (вклучувајќи ќ, ѓ, ѕ, љ, њ, џ) наспроти сите букви (кирилица + латиница). Ако процентот е под 50%, документот се отфрла.
   * Праг: `cyrillic / total > 0.5`

3. **length_filter — Филтер за должина**
   * Ги отфрла премногу кратките документи кои не носат доволно содржина. Минималниот праг е 200 зборови.
   * Праг: `len(doc.text.split()) >= 200`

4. **JsonlWriter — Запишување на резултат**
   * Ги зачувува сите променети документи во JSONL формат во директориумот `clean_data/`.
